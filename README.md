# 📄 Currículo - Maria Vitória Barboza

![Status](https://img.shields.io/badge/Status-Conclu%C3%ADdo-brightgreen)

Este projeto é um currículo desenvolvido utilizando HTML e CSS, como parte de uma atividade da disciplina de Coding Mobile. A versão principal está concluída, mas o projeto poderá receber novas atualizações futuramente para aperfeiçoamento visual e estrutural.

## 📌 Objetivo
O objetivo deste projeto é praticar a criação de páginas web, organizando informações profissionais de forma estruturada e acessível, além de aplicar os conceitos de marcação e estilização vistos em sala de aula.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3

## 📷 Estrutura do projeto
O currículo contém:
- Informações pessoais
- Links clicáveis para contato (email, LinkedIn e GitHub)
- Perfil profissional
- Formação acadêmica
- Certificações
- Tecnologias
- Experiência em projetos

## ▶️ Como executar o projeto
Como este é um projeto estático utilizando apenas HTML e CSS, não é necessário instalar nenhuma dependência. Para visualizá-lo na sua máquina, siga os passos abaixo:

1. Clone este repositório:
   ```bash
   git clone [https://github.com/vitoriaabarbozaa/Curriculo-atv_coding.git](https://github.com/vitoriaabarbozaa/Curriculo-atv_coding.git)
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd Curriculo-atv_coding
   ```

3. Abra o arquivo `index.html` em qualquer navegador da sua preferência.
